== Description ==

This is the unofficial Zettatel plugin for for WooCommerce.

== Features ==

* Send SMS notifications on Woocommerce Order statuses (7)

== Installation ==

* 	Download the zip file from releases
* 	Login to your WordPress Admin.
*   Go to "Plugins > Add New" from sidebar menu.
*   Click on the "Upload" option, then click "Choose File" to select the zip you just downloaded to your device. Once selected, press "OK" and press the "Install Now" button.
*   Activate the plugin.
